Privacy Policy

Effective Date: December 12, 2025

1. Information We Collect

This Farkle Game is a simple multiplayer game. We do not collect personal data such as email addresses, phone numbers, or physical addresses.

  • Game Data: We temporarily store game state (scores, player names you enter) to facilitate multiplayer matches.
  • Discord Data (if applicable): If you access this game via Discord Activity, we may receive your Discord Username and Avatar ID to display in the game. We do not store this permanently.

2. How We Use Data

Data is used solely to functionality of the game, including:

  • Matching players in rooms.
  • Displaying current scores and game status.

3. Data Retention

Game data is ephemeral. Once a game session ends or server restarts, game state is cleared. We do not maintain a permanent database of user history.

4. Third-Party Services

We may use hosting providers (like Railway, Render) which process IP addresses for security and logging purposes.

5. Contact

If you have questions, please contact the developer via the GitHub repository.
